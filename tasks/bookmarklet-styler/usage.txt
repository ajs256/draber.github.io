usage: bookmarklet-styler <source> <target> <stylesheet> [options]

  Inject CSS code into JavaScript code to create a version that can be tested in the browser console.  
  Stylesheets can be any be in CSS, SCSS or SASS format.

Options:

  -m, --marker              Marker in source that should be replaced with CSS, default: bookmarkletCss
  -s, --style               Output style. Can be nested (default), compact, compressed, or expanded.
  -h, --help                Display this help message
  -v, --version             Display the version

Example: 

  bookmarklet-styler js/source.js js/bookmarklet-base.js css/widget.scss -m cssMarker
